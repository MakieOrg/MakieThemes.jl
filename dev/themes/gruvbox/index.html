Gruvbox

The Gruvbox theme[1] is accessible through color_gruvbox_light() and color_gruvbox_dark().

MakieThemes.demofigure(color_gruvbox_light())
MakieThemes.demofigure(color_gruvbox_dark())
MakieThemes.demoscatter(color_gruvbox_light())
MakieThemes.demoscatter(color_gruvbox_dark())
MakieThemes.Gruvbox.color_gruvbox_light — Function
color_gruvbox_light(; kwargs...)

Set up the light version of the Gruvbox color theme.

Keyword arguments:

  • use_alt_bg=true: Use a slightly different background color inside axes, etc.

  • swap_bg=false: Interchange the background colors used for the figure and axes etc.

  • legend_alt_bg=false: Use the alternate background color for the legend background. Which looks better depends on the legend is inside the axis or outside it.

MakieThemes.Gruvbox.color_gruvbox_dark — Function
color_gruvbox_dark(; kwargs...)

Set up the dark version of the Gruvbox color theme.

See color_gruvbox_light for the available keyword arguments.
